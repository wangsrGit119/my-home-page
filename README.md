# Personal Homepage

🏠 Suke的个人主页 - 纯静态个人展示网站

## ✨ 特性

- 🎨 现代设计 - 渐变背景 + 毛玻璃效果
- 🌙 默认暗黑主题，支持主题切换
- 📱 响应式布局，适配桌面和移动端
- ⚡ 纯静态页面，轻量高效

## 🛠️ 技术栈

HTML5 + CSS3 + JavaScript + Animate.css

## 🚀 快速开始

```bash
# 克隆项目
git clone <repository-url>
cd home-page

# 本地预览
python -m http.server 8000
# 访问 http://localhost:8000
```

## 🎨 自定义

- **个人信息**: 编辑 `index.html` 中的姓名、联系方式等
- **头像**: 替换 `static/images/avatar.jpg`  
- **主题色**: 修改 `static/css/custom.css` 中的渐变色

## 🌐 部署

支持 GitHub Pages、Netlify、Vercel 等静态托管服务
